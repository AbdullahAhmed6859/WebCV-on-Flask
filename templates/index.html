Smart Dustbin

This smart Dustbin opens up whenver a person gets nearby. It uses an ultrasonic sensor namely the HC-SR-04, to sense the distance between it and the person in front of it.

Door lock with RFID authentication for Ark Lab

This Door lock uses an electromagnet to keep the door closed. whenever one of the valid RFID tags is touched onto the RFID module, The supply to the electromagnet is cut off for a few seconds and the door opens up. This system ensures only the ARK Core team can access the Lab.

Instagram Follower Bot

This python program is for small businesses and startups or even an aspiring content creator. The idea is to look for a an Instagram page which has the same niche (the type of content we upload on our page) as our Instagram page. The people who follow that page will be the people who are interested in that type of content. So following the folllowers of such a page means there are high chances they will follow us back therefore increasing our reach.

Water Depth measuring device for a water tank

This project was based on the Raspberry Pi, which can be setup on top of an underground water storage or any water tank which has a max depth of 13 feet. It measures the ammount of water remaining and emails the owner when the water level gets way too low. This was according to the demand of the client, so they can order a water tank for their house because they live in an area with an unreliable water supply line.

GUI Password Manager app using Python

When creating a new account on a website, a user can chose to save it on this app. To recall the login details the user can search for the website name on the app. The username and password will then be displayed on a pop up. The app also features a password generator which generates an almost unhackable password. There is also an experimental feature, when used, the app opens the browser and automatically enters all the credentials and lets the user proceed onto the website. This feature is only available for popular websites like Instagram.
